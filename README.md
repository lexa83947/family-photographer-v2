🌐 **Прод:** [taupe-brioche-7aad5d.netlify.app](https://taupe-brioche-7aad5d.netlify.app/)

## Эстетика

**Концепция:** сайт — это не сайт, а **раскрытый плёночный альбом** из 80-х: повёрнутые
полароиды, рукописные подписи маркером, штампы `KODACOLOR · 86`, малярный скотч,
текстура старой фотобумаги. Аналогия — мамин фотоальбом с подписями «Катя, 3 года».

## Типографика

| Шрифт | Где используется | Подключение |
|-------|------------------|-------------|
| **Caveat** (cyrillic) | заголовки, рукописные подписи, кнопки-ссылки | preload + `font-display: swap` |
| **EB Garamond** | основной текст (НЕ Inter) | preload + `font-display: swap` |
| **Special Elite** | штампы, кикеры, метаданные | `font-display: swap` |

## Секции

1. **Hero** — плёночная перфорация сверху, рукописный заголовок, полароид с фото
   (повёрнут, приклеен скотчем), штамп `KODACOLOR · 86`
2. **Обо мне** — портрет в полароидной рамке (повёрнут +3°), текст серифом,
   штампы-достижения `5+`, `200+`, `минск`
3. **Портфолио** — 16 фото в «контактном листе плёнки», каждая карточка
   повёрнута на свой угол; при скролле выравнивается пружинно
4. **Услуги** — два «отрывных билета» с зубчатым краем; правый приподнят
   (featured), с круглой печатью «ПОПУЛЯРНО»
5. **Отзывы** — три «открытки» с наклоном, автопрокрутка каждые 7с
6. **Контакты** — стикеры с контактами + форма в виде **почтовой открытки**
   с маркой и перфорацией; после отправки «письмо улетает»

## Ключевые визуальные приёмы

- Плёночная перфорация (`filmstrip`) сверху и снизу
- SVG-noise текстура бумаги через `feTurbulence` (opacity 0.07, `mix-blend-mode: multiply`)
- Повёрнутые полароиды с пружинным `cubic-bezier(.34, 1.56, .64, 1)`
- Волнистое подчёркивание рукописных заголовков через SVG
- Штампы с двойной рамкой и лёгким поворотом
- Малярный скотч через градиент (полупрозрачный оливковый)
- Зубчатый край билетов через CSS `mask-image`
- Пасхалка: клик по логотипу показывает дату-штамп на всех фото

### Прайс (билеты)

В `index.html`, атрибут `data-price` на элементе `<span>`:

```html
<span data-price="family">300 BYN</span>
<span data-price="women">250 BYN</span>
```

Чтобы поменять цену — отредактировать текст внутри `<span data-price="...">`.

### Отзывы

В `index.html`, атрибуты `data-name` и `data-date` на `<h3>` и `<p class="postcard__date">`,
плюс текст цитаты внутри `<blockquote data-review="N">`:

```html
<article class="postcard" data-name="1" data-date="2025-05-12">
  <h3>Анна Петрова</h3>
  <p class="postcard__date">12 мая 2025</p>
  <blockquote data-review="1">Текст отзыва…</blockquote>
</article>
```

Скрипт берёт `data-name` / `data-date` / `data-review` и копирует в видимые поля —
вам нужно править только разметку, не JS.

### Что оптимизировано

- Все 16 фото конвертированы в WebP (~80 КБ каждое), jpg-фолбэк через `<picture>`
- Hero-портрет имеет `-480` версию для мобильных (`srcset`)
- `<link rel="preload">` для Caveat + EB Garamond
- Google Fonts загружаются асинхронно (`media="print" onload=…`)
- JS минифицирован `terser` и подключён с `defer`
- `loading="lazy"` на всех изображениях кроме hero
- `width`/`height` проставлены — нет CLS
