# Семейный фотограф — Надежда Банцаревич (Минск) · v2 «Плёночный альбом»

Сайт-портфолио семейного фотографа. Стек: **HTML / CSS / JS** (без фреймворков).

## Эстетика

**Концепция:** сайт — это не сайт, а **раскрытый плёночный альбом** из 80-х: повёрнутые
полароиды, рукописные подписи маркером, штампы «KODACOLOR·86», малярный скотч,
текстура старой фотобумаги. Аналогия — мамин фотоальбом с подписями «Катя, 3 года».

## Секции

1. **Hero** — плёночная перфорация сверху, рукописный заголовок, полароид с фото
   (повёрнут, приклеен скотчем), штамп `KODACOLOR · 86`
2. **Обо мне** — портрет в полароидной рамке (повёрнут +3°), текст серифом,
   штампы-достижения `5+`, `200+`, `минск`
3. **Портфолио** — 16 фото в «контактном листе плёнки», каждая карточка
   повёрнута на свой угол; при скролле выравнивается пружинно
4. **Услуги** — три «отрывных билета» с зубчатым краем; средний приподнят,
   с круглой печатью «ПОПУЛЯРНО»
5. **Отзывы** — три «открытки» с наклоном, автопрокрутка каждые 7с
6. **Контакты** — стикеры с контактами + форма в виде **почтовой открытки**
   с маркой и перфорацией; после отправки «письмо улетает»

## Ключевые визуальные приёмы

- Плёночная перфорация (`filmstrip`) сверху и снизу
- SVG-noise текстура бумаги через `feTurbulence` (opacity 0.07, `mix-blend-mode: multiply`)
- Повёрнутые полароиды с пружинным `cubic-bezier(.34, 1.56, .64, 1)`
- Волнистое подчёркивание рукописных заголовков через SVG
- Штампы с двойной рамкой и лёгким поворотом
- Малярный скотч через градиент (полупрозрачный оливковый)
- Зубчатый край билетов через CSS `mask-image`
- Пасхалка: клик по логотипу показывает дату-штамп на всех фото

## Статус формы

Форма-открытка подключена к [Formspree](https://formspree.io) в режиме **заглушки**:
адрес `action` указан как `https://formspree.io/f/your-form-id`, скрипт видит
`your-form-id` и не отправляет письмо, а только логирует данные в консоль
(анимация «письмо улетает» при этом срабатывает).

**Чтобы активировать отправку:**
1. Зарегистрироваться на [formspree.io](https://formspree.io) (бесплатно до 50 писем/мес).
2. Указать `nadya124@inbox.ru` как email получателя.
3. Заменить `your-form-id` в `index.html:485` на свой ID (часть URL после `/f/`).
4. Опционально: включить reCAPTCHA в личном кабинете Formspree (вкладка формы → «Spam protection»).

## Доступность

Автопрокрутка отзывов уважает системную настройку `prefers-reduced-motion`:
у пользователей с включённой опцией «уменьшить движение» таймер не запускается,
открытки остаются на месте. Кнопки «←»/«→» для ручного листания работают всегда.

## Отличия от v1

v1 использовал тёплый скандинавский стиль (Fraunces + Inter, бежевая палитра,
идеально выровненные сетки). v2 — полный редизайн в эстетике «плёночный альбом»
с другой палитрой, шрифтами, ломаной «альбомной» сеткой, декоративными элементами
(скотч, штампы, перфорация) и пружинными анимациями.

## Запуск

Открыть `index.html` в браузере. Никакой сборки не нужно.
