# План: family-photographer-v2 → production-ready

**Цель:** за 2 недели довести сайт-портфолио до состояния, которым не стыдно поделиться с потенциальным клиентом. Целевые показатели: Lighthouse ≥90 по всем категориям, реальная отправка формы, полный контент.

**Допущения:**

## Неделя 2: Производительность + SEO + полировка

### День 6 (Пн) — Производительность: картинки

**Цель:** 16 фото портфолио весят в 3 раза меньше.

| Задача | Файл / место | Критерий «сделано» |
|--------|--------------|---------------------|
| Сконвертировать `family-01..16.jpg` в WebP | `assets/` | Каждое фото ≤80 КБ |
| Оставить fallback в `<picture>` | `index.html:174–255` | `<source type="image/webp">` + `<img src=".jpg">` |
| Добавить `width` и `height` ко всем `<img>` | `index.html` | Браузер резервирует место, нет CLS |
| Добавить `loading="lazy"` ко всему кроме hero | `index.html` | Hero грузится сразу, остальное по скроллу |
| Сжать `photographer.jpg` (если большой) | `assets/photographer.jpg` | ≤200 КБ |

**Инструменты:** `cwebp -q 80 file.jpg -o file.webp` или squoosh.app.

**Ожидаемый эффект:** ~1.7 МБ → ~500 КБ, Lighthouse Performance +5–10 пунктов.
